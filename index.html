主要功能

圈選辨識　Ctrl+Alt+Q

框住公文裡的關鍵字，本機 OCR 辨識後依規則自動把對應批示放進剪貼簿。

詞庫速選　Ctrl+Alt+W

跳出常用擬辦詞庫，搜尋幾個字即時篩選，點一句即複製。

規則與詞庫可自訂

關鍵字、批示、詞庫都能在程式裡新增、編輯、刪除，並可匯入 .txt。

自動學習

OCR 偶爾把字看錯也能對到正確批示，並把誤讀記起來，越用越準。

安裝與使用

1

安裝繁體中文 OCR必做一次

工具靠它讀公文上的字。以系統管理員開啟 PowerShell，確認：

Get-WindowsCapability -Online | Where-Object Name -like "*OCR*zh*"

看 zh-TW 是否 Installed；若是 NotPresent 則安裝：

Add-WindowsCapability -Online -Name "Language.OCR~~~zh-TW~0.0.1.0"
2

啟動

開發／自用：裝 Python 後在資料夾執行 pip install -r requirements.txt，再 python signing_tool.py。

打包成 exe：雙擊 build.bat，會產生 SigningHelper.exe 與一個可分享的 share 資料夾。

⚠請用「以系統管理員身分執行」，全域快捷鍵才會生效。第一次開 exe 跳藍色警告屬正常（沒有付費簽章），按「更多資訊 → 仍要執行」。
3

兩種用法

圈選辨識（Ctrl+Alt+Q）：畫面變暗後拖曳框住關鍵字 → 命中後右下角顯示「已複製」→ 到欄位按 Ctrl+V。

詞庫速選（Ctrl+Alt+W）：跳出詞庫，搜尋後點一句即複製，再 Ctrl+V。

＋公文字太小讀不到：把檢視器顯示放大到 150~200%（Ctrl+滑鼠滾輪），或用 Windows 放大鏡（Win++），再框選最有效。
